Social Overview

Figure 1. The Social Screen

  1. The Filters & Search panel. This allows you to filter the reviews.
  2. The Reviews panel. This shows all reviews left by users.

The Social screen can be accessed from the social button on the navbar. This screen allows you to see all of the reviews left by other users.

Filtering

Using the Filters & Search panel, you can filter the reviews shown by wine, user, or rating. You can also search for a user using the button at the top.
