Raw Data Overview

Figure 1. The Raw Data Viewer

  1. The Filters panel. THis is the same as the one on the standard wine view, and your filters will persist across these two pages.
  2. The Table panel. This shows all the data in a compact table view.

This is the raw data view screen. It lets you view all the data in the database in a compact table format. Like the standard wine view, you can open the detailed view by double-clicking on an entry. This view also uses the same paging system as the standard wine view.
