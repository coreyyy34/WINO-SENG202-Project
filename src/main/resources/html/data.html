Importing Data

You can import data in the .csv format. Once you picked your file by clicking the 'Open CSV' button, you will be shown the columns in the CSV:

Figure 1. The import screen after selecting a CSV

Here, you need to use the dropdown menus to match each CSV column to a database column. Note that only the TITLE column is required, but you should aim to match as many as possible for a complete experience. To add the dataset, you can either REPLACE or APPEND with the buttons in the top-right.
